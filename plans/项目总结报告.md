## 🔧 技术架构详情

### 核心模型架构
```
EnhancedPlateRecognitionModel
├── EfficientResNetFeatureExtractor
│   ├── ResNet50特征提取
│   ├── CBAM注意力模块
│   └── 通道降维
├── MultiScaleFeatureFusion
│   ├── 1x1卷积
│   ├── 3x3卷积
│   └── 5x5卷积融合
├── Transformer编码器
│   ├── 6层编码层
│   ├── 8头注意力
│   └── GELU激活函数
└── 输出层
    ├── 字符分类器
    └── 类型分类器
```

### 性能指标
- **模型参数量**: 36,351,015
- **字符类别数**: 78个（包含特殊字符）
- **车牌类型数**: 7种
- **推理时间**: < 50ms
- **支持分辨率**: 224x224

## 🚀 部署指南

### 1. 环境准备
```bash
# 安装依赖
pip install -r requirements.txt

# 创建日志目录
mkdir -p logs
```

### 2. 训练模型
```bash
# 标准训练
python train_enhanced.py --epochs 100 --batch_size 64

# 快速测试
python train_simple.py
```

### 3. 启动服务
```bash
# 启动API服务
python deployment_service.py

# 服务地址: http://localhost:8000
```

### 4. 性能分析
```bash
# 生成性能报告
python performance_monitor.py
```
